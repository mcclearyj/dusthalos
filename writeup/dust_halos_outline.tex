\subsection{Correlation Function Algorithm\label{ssec:treecorr}}
\begin{outline}
\1 Treecorr algorithm, briefly (not our algorithm) but it is what we use. Correlating not gal-shear but gal-extinction
	TreeCorr\footnote{https://github.com/rmjarvis/TreeCorr} is a package for efficiently computing 2-point and 3-point correlation functions. 
	\2 cite Jarvis, Bernstein, \& Jain, 2004, MNRAS, 352, 338
	\2  Among others, the package returns the two-point number counts-$\kappa$ correlation function, where $\kappa$ can be any scalar quantities. To measure extinction, we substitute the value of 		the reddening vector for $\kappa$ and use the position of the foreground sample as number counts
	\2 Mention spatial correlation as well: two-point correlation function of number counts (typically galaxy counts)

\1 Mention null tests: flexibility of algorithm allows us to easily correlate any two samples, so ideal for null tests like star-extinction

\end{outline}
